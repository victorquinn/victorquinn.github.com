DuckDuckGo Safari Extension Update 1.3

Aug 27th, 2010

Download

This is another mostly style update for the DuckDuckGo Safari Extension. First, a screenshot of Version 1.2:

The first thing I did was switch from the square DDG icon to the new round one. I do my best to stay current and in sync with DuckDuckGo and a favicon update there dictates one here.

I also made some other style changes to match up with them a bit better. The previous versions of the bar had rounded edges because that is the way Apple differentiates between the URL and the Search bar. If you look at Safari on the desktop or on any iOS device, you will notice that the URL has squared off edges whereas the search box has very rounded edges. This is an easy way for the user to differentiate. However, since the DuckDuckGo search cannot go in the top, I decided to opt for something that more closely matches the DuckDuckGo search rather than the built in Safari search. Again, many would argue this is excessive attention to detail, but I wanted it to be perfect. 

I also replaced the regular search button that said “DuckDuckGo!” with a more professional looking image search button using the same image that DuckDuckGo uses. This gives further consistency and looks quite a bit nicer in my opinion.

And last, since I removed DuckDuckGo! text from the search button, I still wanted it to say the name somewhere. So I pre-populate the textfield with that text in a light grey. I make it disappear when focus is granted to the textfield and also return the textfield to a normal color so the search terms stand out.

I also ensured that the search bar looked alright in Windows. Previously, I had developed it on a Mac so I did not see how poorly it looked on Windows. With this update, I made sure to make it look equally nice and pixel perfect on Safari for Windows.

The new look:

The extension is now listed on Apple’s Safari Extension Gallery.
